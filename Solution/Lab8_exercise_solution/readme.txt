This is an example of Lab 8 exercise solution.

1. You need to go into ScalableML directory, which has been downloaded from GitHub.

2. Put .py file into Code folder

3. In the .py file, change your username in the SparkSession builder part.

4. Put Lab8_Exercise.sh into HPC folder.

5. Then 'cd HPC' folder and run 'sbatch Lab8_Exercise.sh'

5. Output file will be in the Output folder.
